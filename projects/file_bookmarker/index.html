File bookmarker

Overview

ファイルをブックマークして素早く開くことができるようにするソフトです。
現在はWindowsのみのリリースとなります。

Download

Usage

ファイルをブックマークする方法

①ソフトを起動しブックマークフォルダを選択(デフォルトではbookmark)
②ファイルをドラッグドロップする



ブックマークを見る方法

①ファイル一覧をクリック
②ブックマークフォルダを選択するとピン止めされたファイル一覧を見ることができます



新しくブックマークフォルダを作る方法

①＋ボタンをクリック
②新規ブックマークフォルダの名前を入力する
