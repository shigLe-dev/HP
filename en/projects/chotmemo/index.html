Usage

How to take note

You can write a note by typing it in the text box and pressing the Enter key
If you want to start a new line, press the SHIFT key and the Enter key at the same time.

You can save the image by pressing the button in the lower left corner.



How to make a new file.

Please type a new file name in the left text box and click the right plus button.



How to search notes


You can search by typing a word that you want to search and press the ENTER key.



How to pin notes


You can pin the note by click the upper right button of pin mark.
You can see pinned notes list by click the upper right button of pin mark in window.
