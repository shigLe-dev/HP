Chot Memo

Overview

チャットを使っているような感覚でメモができるメモ帳です。
気軽に書くことができ、インストールなしで使うことができます

Application page

Usage

書き込む方法

テキストボックスの中にメッセージを入力してEnterキー(改行)を押すことで書き込むことができます。
改行したい場合はshiftキーとEnterキーを同時に押してください。

左下のボタンを押すことで画像を書き込むことができます。



新しいファイルを作成する

左のテキストボックスに新規ファイル名を入力して、右の＋ボタンを押します



過去の書き込みを検索


上の検索ボックスに検索したい言葉を入力し、Enterキーを押すと検索することができます



書き込みをピン止め


書き込みの右上にあるピン止めボタンを押してピン止めします
画面の右上のピンマークでピン止め一覧を見ることができます
