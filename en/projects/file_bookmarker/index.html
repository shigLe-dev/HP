Usage

How to bookmark files.

①Please start this software and select a bookmark folder.(Default: bookmark)
②And Drag and Drop the file in this box.



How to open bookmarked files.

①Please click the files list.
②And select a bookmark folder,You can see the files list.



How to make a new bookmark folder.

①Please click the "＋" button
②And type a new file name.
